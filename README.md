# How to Install

Setup your environment as per the [React Native](https://reactnative.dev/docs/set-up-your-environment) docs.

```shell
 git clone git@github.com:InvisOn/Nutrient-Tracker.git
 cd Nutrient-Tracker/
 npm install
 npx expo start
```
